#  ValenorManager.Domain

Camada responsável pelas **regras de negócio** do sistema Valenor Manager.

---

##  Sobre o Projeto

O `ValenorManager.Domain` representa o **coração da aplicação**, contendo:

- Entidades de negócio
- Regras e validações
- Comportamentos do sistema
- Exceções específicas de domínio

Essa camada é **independente** de frameworks e infraestrutura.

---
