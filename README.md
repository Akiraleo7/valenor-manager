# Valenor Manager

Sistema de gestão de estoque e vendas desenvolvido para o PIM (Projeto Integrado Multidiciplinar) do curso de Análise e Desenvolvimento de Sistemas da Universidade Paulista (UNIP).

## Estrutura do projeto

- backend → lógica e API
- frontend → interface do usuário
- database → modelagem do banco de dados
- docs → documentação do projeto

## Tecnologias
- Backend: ASP.NET
- Banco de Dados: MySQL
- Frontend: HTML, CSS e JavaScript

## Objetivo

Fornecer uma solução completa para controle de produtos e vendas.
